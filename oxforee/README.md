# Oxford 3000 Vocabulary Premium 📚

เว็บแอปพลิเคชันสำหรับฝึกฝนคำศัพท์ Oxford 3000 พร้อมฟีเจอร์ช่วยจำมากมาย

## ฟีเจอร์หลัก (Features)
- **Dashboard**: แสดงสถิติคำศัพท์ทั้งหมด
- **Flashcards Review**: โหมดทบทวนคำศัพท์แบบบัตรคำ (หน้า/หลัง)
- **Text-to-Speech**: ฟังเสียงอ่านคำศัพท์
- **Auto-fetch**: ดึงคำแปลและตัวอย่างประโยคอัตโนมัติจาก Dictionary API
- **Import/Export**: รองรับการโหลดข้อมูลคำศัพท์จำนวนมาก

## วิธีใช้งาน (Usage)
1. เปิดไฟล์ `index.html` เพื่อเริ่มใช้งาน
2. กดปุ่ม **Load Data** เพื่อโหลดคำศัพท์พื้นฐาน
3. กด **Review** เพื่อเริ่มทบทวน

## การติดตั้งบนมือถือ (Mobile)
